[Universe] Signal Wow (Extraterrestre !?)

The Wow signal is a radio signal received in 1977.

The SETI project aims to listen to the sky in order to possibly hear extra-terrestrial radio waves. In summer 1977 a station will pick up this signal which seems to be artificial (we have no recording, all we have is this frequency.)

Thus, this stop allows to talk about extra-terrestrials without falling into the conspiracy.
